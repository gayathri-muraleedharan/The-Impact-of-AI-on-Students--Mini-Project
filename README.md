# The-Impact-of-AI-on-Students--Mini-Project
This mini project analyses the impact of Generative Artificial Intelligence (GenAI) on students' academic performance, study habits and wellbeing using a dataset of 50,000 student records. Microsoft Excel was used for data cleaning, data transformation, exploratory data analysis and the development of an initial dashboard. Microsoft Power BI was then used to create an interactive business intelligence report featuring KPI cards, DAX measures, slicers, navigation buttons and dynamic visualisations. The project demonstrates the complete data analytics workflow, from preparing raw data to generating meaningful insights that support data-driven decision-making.
